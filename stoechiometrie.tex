\section{Stöchiometrie chemischer Reaktionen}
\subsection{Allgemeines}
$N$ Komponenten: $A_1...A_N$ chemische Spezies\\
Schlüsselkomponenten: Moländerungen müssen bekannt / messbar sein, um eine Aussage über die Moländerungen der anderen Komponenten zu bekommen.
Schlüsselreaktionen:
\[ \frac{dc_i}{dt} = \sum_{j=1}^{M} \nu_{ij} r_j \]
Anzahl der Mole eines Elements:
\[ b_h = \sum_{i=1}^N \beta_{hi} n_i \quad \sum_{i=1}^N \beta_{hi} \Delta n_i = 0\]

Anzahl der Spezies meist größer als Anzahl der Elemente. Dann miest $R_{\beta} = L$.\\
\[ R = N - R_{\beta} \]
\begin{itemize}
\item $R$ Zahl der Schlüsselkomponenten - Freie Unbekannte
\item $R_{\beta}$ gebundene Unbekannte - werden berechnet
\item Anzahl Schlüsselreaktionen = Anzahl Schlüsselkomponenten
\end{itemize}
Das System kann von unten gelöst werden: $\Delta n_{H_2} = - \Delta n_{CO} - 2 \Delta n_{CO_2}$ etc.

\subsection{Beziehungen zwischen Stöchiometrie und Reaktionskinetik}
Um sinnvolle Erkenntnisse aus den Berechnungen zu erhalten, welche den Reaktionsverlauf oder Reaktionsmechanismus beschreibt, ist eine Bewertung notwendig.

\begin{enumerate}
\item $M = R_{\nu} = N - R_{\beta}$ - Anzahl der wirklichen Reaktionen entspricht der Anzahl der Schlüsselreaktionen.\\
  Bsp. 3 Isomere, 1 Schlüsselreaktion
  \begin{itemize}
     \item Parallelreaktion ($A_1 \rightarrow A_2$ und $A_1 \rightarrow A_3$)
     \item Folgereaktion ($A_1 \rightarrow A_2 \rightarrow A_3$)
  \end{itemize}
  Entscheidung durch Konzentrationsmessung von $A_3$ an $t=0$. Wenn $\frac{dc_3}{dt}=0$, dann 2.
\item $M > R_{\nu} = N - R_{\beta}$ - Mehr ablaufende Reaktionen als Schlüsselreaktionen
\item $M < R_{\nu} = N - R_{\beta}$ - Eine oder mehrere Schlüsselreaktionen sind kinetisch unmöglich.\\
  Bsp. Rohrzuckerinversion.
\end{enumerate}
